# Trae Theme for VS Code & Cursor

这是一个为VS Code和Cursor编辑器设计的主题扩展。提供了以下主题选项：

- Dark：深色主题
- Deep Blue：深蓝色主题
- Light：浅色主题

## 安装

### 从VSIX安装
1. 下载最新的发布版本
2. 在VS Code/Cursor中，选择"从VSIX安装..."选项
3. 选择下载的VSIX文件

### 通过命令行安装
```bash
# VS Code
code --install-extension icube-themes-1.0.0.vsix

# Cursor
cursor --install-extension icube-themes-1.0.0.vsix
```

## 应用主题
1. 打开命令面板 (Cmd+Shift+P 或 Ctrl+Shift+P)
2. 输入 "Color Theme"
3. 选择一个Trae主题
